Java 高级工程师

首页   >   加入我们   >   岗位招聘
工作职责

1. 负责互联网产品架构分析, 设计满足当前及未来运营指标的软件架构

2. 预研主流先进的高可用，高扩展，高性能技术架构，并应用于产品开发

3. 驱动核心技术问题的攻关, 系统优化, 协助解决项目开发过程中的技术难题

4. 编写技术架构规范, 制定技术标准，约定代码规范

5. 协调制定整体开发计划,　对团队技术人员进行关键技术培训和指导

6. 协助管理者对团队开发人员进行效率考核

任职要求

1. 3年以上一线软件项目开发经历

2. 具备架构师所需计算机基础理论，网络知识，Linux操作系统管理及优化知识

3. 了解负载均衡，反向代理，缓存redis/memcache，互联网安全等整体架构层相关知识

4. 精通Mysql 主从/主主 架构设计, 至少掌握一种 nosql数据库：MongoDB/Cassandra/HBase

5. 精通 Java语言,JavaEE规范及JVM性能参数调整

6. 掌握利用消息系统如 RabbitMq, ActiveMq 的异步接口模型设计

7. 有互联网系统架构设计经验优先

8. 了解 WEB 前端框架 BootStrap/AngularJs/Backbone/Knockout 优先

9. 良好的英文阅读能力

10. 主动性及团队协作能力
